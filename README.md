📚 Bibliotech

O Bibliotech é uma mini biblioteca digital desenvolvida com o objetivo de facilitar o gerenciamento de livros, empréstimos e devoluções. O projeto foi construído de forma simples, intuitiva e funcional, utilizando apenas Python e Streamlit para criar uma interface acessível e prática.

🚀 Funcionalidades

🔸 Cadastro de livros (título, autor, categoria)
🔸 Registrar empréstimo (nome da pessoa + livro)
🔸 Marcar devolução

📈 Evolução planejada

⏳ Filtro por categoria
⭐ Indicar livros mais emprestados
⚠️ Aviso de atraso (baseado em dias de empréstimo)

🛠️ Tecnologias Utilizadas

- Python

- Streamlit

Essas ferramentas permitem a criação de interfaces rápidas e fáceis de usar, além de possibilitar o processamento dos dados de forma eficiente.
